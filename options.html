API Key Settings

This extension uses free Alpha Vantage APIs to retrieve stock data. To use these APIs, you must claim your own API key(s). Visit the Alpha Vantage website to obtain them. Then fill in the fields below with your key(s), each of which should be a long string of numbers and letters.

Note that while 1 key is the minimum number required to use this extension, each key allows only a limited number of API calls per minute. As such, generating 2 keys with 2 separate emails will allow the extension to return information more reliably.
