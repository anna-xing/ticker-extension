Set your API key(s)

This extension uses the free Alpha Vantage API to retrieve stock data. To use it, you must claim your own API key(s). Visit the Alpha Vantage website to obtain them, then fill in the fields below. Each key should be a long string of numbers and letters.

You need at least 1 key to use this extension. However, note that each key allows only a limited number of API calls per minute. This means that the extension will return information more reliably if you generate 2 keys with 2 separate emails.
